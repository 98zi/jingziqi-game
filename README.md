# 基于react框架的井字棋游戏

## 项目背景
井字棋游戏，该游戏基于react框架，来源于react官方文档，我在学习的过程中把这个小游戏也做了一遍

## 使用方式
使用它?
```shell
git clone https://github.com/anghunk/jingziqi-game
```

进入项目？
```shell
cd jingziqi-game
```

运行它？
```shell
npm start
```

可以打开浏览器使用它


## 打包

```shell
npm run build
```

打包完成之后，在dist文件夹内点击index.html即可使用
